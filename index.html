What is Intecture?

Intecture is a developer friendly, language agnostic configuration management tool for server systems.

  • Extensible support for virtually any programming language
  • Standard programming interface. No DSL. No magic.
  • Rust API library (and bindings for popular languages)

Quick Start Guide

The easiest way to install and configure Intecture is via the interactive installer:

$ curl -sSf https://static.intecture.io/install.sh | sh

You can also compile and configure each component separately. Instructions for manual installation can be found in the README.md file in each repository.

Once you've installed Intecture, it's time to create your first project and get coding!

Use the CLI to create a new project and install the example framework.

$ cd /path/to/my-projects $ incli init my-project php Created project my-project

That wasn't so hard. Now let's run it!

$ cd my-project $ incli run Usage: incli run <server_host_or_ip>

Ok, so that wasn't very exciting, but let's dig into it. If you look in your my-project directory, you'll see a bootstrap file. For PHP projects, it's bootstrap.php. In this file you'll see that it's asking for an argument <server_host_or_ip>. This refers to the host you want Intecture to build (remember that this host must run the Intecture Agent).

Let's try it again with the server IP as an argument. Remember that your IP will probably not be 127.0.0.1 as in the example.

$ cd my-project $ incli run 127.0.0.1 Array ( [exit_code] => 0 [stdout] => root [stderr] => )

That's more like it! You've just run your first remote command. Make sure you check out the documentation: docs.intecture.io.

Components

Intecture aims to be a fully distributed, service orientated system. Thus it is broken up into several modules.

CLI

intecture-cli

The CLI lives on developers' machines and is the primary interface with Intecture. Use it to kick off builds and manage the system.

API

intecture-api

The API is the interface between your code and the Intecture Agent on your managed hosts.

Bindings

intecture-api-c
intecture-api-php
intecture-api-ruby

Agent

intecture-agent

Each server that you want to manage must run the Intecture Agent. This service listens on a user-defined TCP socket for incoming requests from the API.
